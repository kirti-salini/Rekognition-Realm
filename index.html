This website allows you to upload images and receive object recognition results delivered to registered emails. It's a platform where users can securely upload their images, and upon submission, the system processes these images using AWS Rekognition services. The results are then sent to the user via email for their records and convenience.

How it Works:

  1. Choose a file to upload in the Upload Section.
  2. Click the "Upload" button to send your file to the server.
  3. Receive an email with object recognition results once processing is complete.
